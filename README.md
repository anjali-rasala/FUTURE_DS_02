# FUTURE_DS_02
Customer Retention &amp; Churn Analysis using Power BI

## Objective
Analyze customer churn patterns and retention behavior using Power BI to identify the major factors influencing customer loss and provide actionable business insights.

## Tools Used
- Power BI
- CSV Dataset
- Data Cleaning & Visualization
  
## Dataset Used
Telco Customer Churn Dataset

## Dashboard Features
- Total Customers KPI
- Churned Customers KPI
- Average Monthly Charges KPI
- Churn Rate KPI
- Churn Distribution Analysis
- Churn by Payment Method
- Churn by Contract Type
- Churn by Internet Services
- Monthly Charges vs Churn
- Interactive Slicers

## Key Insights
- Customers with month-to-month contracts have higher churn rates.
- Fiber optic internet users showed higher churn behavior.
- Customers with higher monthly charges are more likely to churn.
- Electronic check payment method had more churned customers.
- Overall churn rate is approximately 26.58%.

## Recommendations
- Provide better retention offers for month-to-month customers.
- Improve customer support for fiber optic users.
- Introduce loyalty discounts for high monthly charge customers.
- Encourage long-term contracts to improve retention.
